# CRYPTOGRAPHY PROJECT: HAHACOIN
We create a simple blockchain system with our new Crypto Coin name: HaHaCoin

## Installation
Clone the project
```
git clone https://github.com/Hieunm54/Cryptography_project.git
```

## Run the project
* Run the backend

* Run the fronted
```
cd frontend
npm start
```
